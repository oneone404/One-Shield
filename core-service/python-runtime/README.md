# Python Runtime Directory

Thư mục này sẽ chứa Python portable runtime.

## Setup (Optional)

Nếu muốn bundle Python vào ứng dụng:

1. Download Python Embeddable từ: https://www.python.org/downloads/windows/
2. Giải nén vào thư mục này
3. Cài đặt pip: `python get-pip.py`
4. Cài đặt PyTorch: `pip install torch --target .`

## Alternative

Hoặc sử dụng Python đã cài trên hệ thống người dùng.
